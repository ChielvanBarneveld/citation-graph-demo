# FORAS citation-graph demo (v5)

**Live:** https://chielvanbarneveld-citation-graph-demo-streamlit-app-wfx5qe.streamlit.app/

Four-tab Streamlit app for the FORAS PTSD-trajectories review experiment:

- **FORAS** - 11.873-node citation graph (172 FT-included core, 395 TIAB, ~7.6k screened, ~3.7k external)
- **ASReview** - active-learning baseline + 70/30 vs GCN comparison
- **GNN** - learning-lab + over-smoothing demo + sentinel-rangen
- **Candidates** - 2.288 historical-terminology kandidaten + drie sentinel-cards (Solomon 1993 EASY, Kardiner 1941 MEDIUM, Southard 1920 HARD)

Sidebar control panel: view-mode toggle (Leerlab / Demo), focus-sentinel selector, bookmarks.

Source: https://github.com/ChielvanBarneveld/citation-graph-demo
